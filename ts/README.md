## Query SDL

|symbol|mean|
|:----:|:----:|
| `=`| `equal`|
| `<>`| `not equal`|
| `;`| `terminator`|
| `>`| `great than`|
| `>=`| `great than or equal`|
| `<`| `less than`|
| `<=`| `less than or equal`|

example

```
star:>100  this mean: the star number more than 100
age:<20   this mean: the age number more than 100
```
